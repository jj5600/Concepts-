week 1 overview:
Welcome to the amazing world of Embedded Controller Programming with Embedded C ---  the technology foundation for the ever growing number of embedded devices --- including the fast growing IoT (Internet of Things) devices!

NOTE! NOTE! NOTE!  Since this lesson was created the new STM32CubeIDE has been released,  which integrates the individual tools listed below. The info below is still valid except that instead of separate tools they are now integrated into a single STMCube32IDE.

In this lesson, you are going to accomplish three major goals:

Goal #1. An introduction to Embedded C, and how it is different from standard C.

Goal #2. An introduction to STM32CubeMX, a GUI (graphical user interface) tool that we will use to configure I/O pins and other features, then generate initial embedded C code for our STM32L Discovery Kit IoT Node.

Goal #3. An introduction to TrueStudio IDE (Integrated Development Environment), a key tool we will use to write, build, and run our Embedded C code.

NOTE! NOTE! NOTE! The new STM32CubeIDE combines both STM32CubeMX + TrueStudio.

At the end of this lesson, you will have installed STM32CubeIDE and have writtenyour first embedded C code to blink the LED on the STM32L4 Discovery Kit board! 

Preview:

Since many of you will be ordering your STM32 Discovery Kit IoT Node during Lesson 1, you will be able to complete your Lesson 1 programming assignment with or without the board. 

If you have already received your STM32 Discovery Kit IoT Node, take a look at the STM32L476 MCU that is on the board. Notice all the many pins on the chip. And imagine that each of the pins can be configured different ways: GPIO Input, GPIO Output (with or without pull-up resistor), Serial I/O, and more.

Think about how hard and it would be to configure the chip, and create the C initialization code, without the proper tools.

What type of GUI-based tools can you imagine would be useful to help configure the I/O pins, and also generate C code? And what type of IDE to use once the code is generated?
